How To

Purpose

Digital Detective v1.6 helps you inspect, capture, and troubleshoot web pages by extracting HTML, generating design system reports, taking screenshots, and identifying scripts.

Tools

  • Screenshot — Capture a component, the visible viewport, or the full page.
  • Color Picker — Pick a color and copy its HEX value.
  • Element Picker — Click any element to copy its outerHTML.
  • Event Tracker — Inspect Segment/Rudderstack events fired from the page.
  • Component Blueprinter — Paste HTML and generate a structured component report.
  • DS Extractor — Generate a design system report for the full page or a selected component.
  • Match Analysis — Compare two URLs side by side and get a visual diff report with a match score per category.
  • Lighthouse Audit — Run a Google Lighthouse performance report on any public page and view Core Web Vitals, opportunities, and diagnostics.
  • SEO Analysis — Audit on-page SEO signals across 5 pillars (indexability, on-page, content quality, semantic SEO, crawlability) and get a failure-first scored report.
  • Script Finder — List external scripts, copy their code, and enable/disable them.
